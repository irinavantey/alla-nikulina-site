# Сайт Аллы Никулиной

Готовая статическая версия сайта находится в этой папке.

## Как загрузить на сервер

1. Загрузите содержимое папки `site` в корень сайта на хостинге.
2. Стартовая страница: `index.html`.
3. Дополнительные страницы: `about.html`, `formats.html`, `resilience.html`, `resources.html`, `film.html`.
4. Все локальные изображения, PDF, CSS, JS и видео лежат в `assets`.

## Важно про видео

Видеофайлы занимают около 2 ГБ. Для быстрой загрузки сайта лучше перед публикацией:

- сжать видео до web-версий;
- или загрузить их на VK Video / YouTube / Rutube и заменить локальные `<video>` на встраиваемые плееры.

Сайт работает и без сборщика: Node.js, PHP и база данных не нужны.
