# Рабочая документация проекта

Проект: сайт Аллы Никулиной, коуча для предпринимателей, руководителей и первых лиц компаний.

Цель сайта: вызывать доверие, спокойно объяснять подход эксперта и мягко вести пользователя к сессии-знакомству, индивидуальной работе, executive-коучингу или программе "Жизнестойкость лидера".

## Где что лежит

- `site/` - основная рабочая версия сайта. Здесь лежат страницы, локальные видео, изображения, PDF и исходная статическая версия.
- `docs/` - копия сайта для GitHub Pages. В ней видео убраны из `src`, чтобы не пушить тяжелые `.mp4` на GitHub.
- `док/` - эта папка с рабочей документацией для следующей сессии.
- Исходные материалы клиента лежат в корне проекта и рабочих папках рядом, но не отслеживаются Git.

## Важное про видео

Исходные видео весят около 2 ГБ и исключены из Git через `.gitignore`.

Локальная версия `site/` использует видео из:

- `site/assets/video/intro.mp4`
- `site/assets/video/coaching.mp4`
- `site/assets/video/self-regulation.mp4`
- `site/assets/video/mountain-meditation.mp4`

В `docs/` видео не загружены в GitHub Pages-версию. Для публичного сайта нужно:

- либо сжать видео в web-версии;
- либо загрузить видео на VK Video / YouTube / Rutube и заменить локальные `<video>` на embed.
