# Быстрый старт следующей сессии

## Что сделали

1. Собрали статический многостраничный сайт эксперта-коуча.
2. Использовали материалы из ТЗ, прототипа, статьи, пакетов услуг, фото, видео и гайдов.
3. Создали страницы:
   - `index.html` - главная;
   - `about.html` - об эксперте;
   - `formats.html` - форматы работы;
   - `resilience.html` - программа "Жизнестойкость лидера";
   - `resources.html` - бесплатные материалы;
   - `film.html` - фильм / личная стратегия.
4. Подключили Git и GitHub.
5. Добавили папку `docs/` для публикации через GitHub Pages.

## Основные дизайн-решения

- Основной фон светлый.
- Акценты: насыщенный зелено-бирюзовый и желтый.
- Темные блоки имеют легкий зелено-бирюзовый градиент.
- Акцентный шрифт: `Unbounded`.
- Основной шрифт: `Gotham Pro`, с fallback на `Gotham`, `Arial`, `sans-serif`.
- Карточки с радиусом `8px`.
- Меню использует базовые цвета: зелено-бирюзовый и желтый.

## Важные UX-решения

- Главная ведет через состояние клиента, а не через прямую продажу.
- CTA разные:
  - часть открывает форму заявки;
  - часть ведет напрямую в Telegram.
- Форма заявки создается через `site/assets/js/main.js`.
- Форма не имеет backend: она копирует текст обращения и открывает Telegram.
- Короткие предлоги и союзы защищены от висячих переносов JS-типографикой.

## Видео

Вертикальные видео оформлены через класс `.phone-video`.

Горизонтальные видео остаются обычным широким плеером `.video-box`.

Последняя правка убрала большой зеленый фоновый прямоугольник вокруг айфона.

Если продолжать полировку видео:

- проверить реальные постеры;
- решить, какие видео должны быть вертикальными, а какие горизонтальными;
- заменить локальные видео в `docs/` на embed или сжатые файлы.

## Что проверить в следующую очередь

1. Открыть GitHub Pages после выбора `main` + `/docs`.
2. Проверить, как сайт выглядит без видео в публичной версии.
3. Если нужны видео на публичном сайте, выбрать способ:
   - VK Video / YouTube / Rutube;
   - сжатые web-видео;
   - отдельный хостинг для видео.
4. Визуально пройти мобильную версию:
   - шапка;
   - меню;
   - телефонные видео;
   - плавающая CTA-плашка;
   - форма заявки.
5. Уточнить реальную Telegram-ссылку эксперта. Сейчас используется:
